# Base UI 组件库

基础 UI 组件库，包含 Button、Input、Select 等基础组件。

## 安装

```bash
npm install @regan-ad/base-ui
```

## 使用

```tsx
import { Button } from '@regan-ad/base-ui';

export default () => <Button>Click me</Button>;
```

## 组件列表

### 基础组件

下面列出所有可用的基础组件：

- **Button 组件** - 按钮组件，用于触发操作
- **Input 组件** - 输入框组件，用于用户输入
- **Select 组件** - 选择框组件，用于选择选项

> 💡 **提示**：点击左侧菜单中的各个组件名称即可查看详细文档、示例代码和 API 文档。
